Djangoni tezkor o'rnatish qo'llanmasi

Djangodan foydalanishingiz uchun avval u o'rnatilgan bo'lishi kerak bo'ladi.Barcha imkoniyatlarni o'z ichiga olgan to'liq o'rnatish qo'llanmasini bu yerda o'qishingiz mumkin.Ushbu qo'llanma esa sizga djangoni minimal va tezkor usulda o'rnatishingiz uchun sizga yo'l ko'rsatadi.

Django Python web-frameworki bo'lgani uchun u Python dasturlash muxitini talab qiladi.Pythonning qaysi versiyalarini Django bilan ishlatish mumkinligini ko'rib chiqishingiz mumkin. Ushbu Python versiyalari SQLite ma'lumotlar bazasini o'z ichiga oladi, Shuning uchun Django bilan tanishish uchun ma'lumotlar bazasini o'rnatishingiz shart emas.

Pythonning eng so'nggi versiyasini bu yerdan https://www.python.org/downloads/ ko'chirib olishingiz yoki operatsion tizimi paket menejeri yordamida uni o'rnatish mumkin.Yoki Pythonni o'rnatishni ko'rib chiqishingiz mumkin.

Python o'rnatilganligini tekshirish mumkin terminalda python buyrug'ini bajarish kerak:

 $ python3

 Python 3.7.5 (default, Nov 20 2019, 09:21:52) 
 [GCC 9.2.1 20191008] on linux
 Type "help", "copyright", "credits" or "license" for more information.
 >>>

Ma'lumotlar bazasini o'rnatish

Ushbu qadam faqat PostgreSQL, MariaDB, MySQL yoki Oracle kabi "katta" ma'lumotlar bazasi dvigateli bilan ishlashni istasangiz kerak. Bunday ma'lumotlar bazasini o'rnatish uchun, ma'lumotlar bazasini o'rnatish usullari bilan tanishing.

Djangoni o'rnatish

Pip yordamida o'rnatish.

Eng oson qulay yechim pip yordamida o'rnatish hisoblanadi.

  • Buning uchun sizda pip o'rnatilgan bo'lishi kerak ,agar u eskirgan bo'lsa, yangilash maslaxat beriladi. Chunki pip yangilanmasa Djangoni o'rnatish ishlamasligi mumkin.

  • Virtualenv va virtualenvwrapper,kabi narsalarga xam bir ko'z tashlang. Bu vositalar boshqa 'narsalardan' aloxida 'izolatsiyaga' olingan Python muhitlar yaratib beradi. Bundan tashqari, administrator imtiyozlarisiz paketlarni o'rnatish imkonini beradi. Ushbu qo'llanma yordamida virtualenv yaratish xaqida ko'proq ma'lumot olishingiz mumkin.

  • Virtual muhitni yaratganingizdan va faollashtirganingizdan so'ng buyruqni kiriting:

 

 $ python -m pip install Django

Bu buyruq Djangoning eng oxirgi versiyasini o'rnatadi.Agar sizga Djangoning aynan qaysidur bir versiyasi kerak bolsa unda ushbu buyruqdan foydalanishingiz mumkin

 

 $ python -m pip install Django==2.2.7

Ikkita tenglik belgisidan so'ng kerakli versiya ko'rsatiladi

Djangoni o'rnatilganligini tekshirish

Django Python uchun ishlatilishga tayyorligini tekshirish uchun terminaldan python muxitiga kirib djangoni import qilamiz

$ python3

Python 3.7.5 (default, Nov 20 2019, 09:21:52) 
[GCC 9.2.1 20191008] on linux
Type "help", "copyright", "credits" or "license" for more information.
>>> 
>>> import django
>>> print(django.get_version())
3.0

!Sizda Djangoning boshqa versiyasi o'rnatilgan bo'lishi mumkin.
